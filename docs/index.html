    How it works

    1. Fetch from GitHub

      Reads issues, pull requests, and comments through the GitHub API.

    2. Render Markdown

      Converts repository data and prepares optional raw JSON.

    3. Download ZIP

      Creates a portable archive you can keep, search, and share.
